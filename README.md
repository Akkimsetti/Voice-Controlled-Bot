# Voice-Controlled-Bot


### Materials Required
- Nodemcu or any other board which support ESP8266
- L298N Motor Driver
- 2 Gear Motors
- 2 3.7v Lithium Ion Batteries
- IFTTT

The bot can be controlled wirelessly with the help of Voice Commands
